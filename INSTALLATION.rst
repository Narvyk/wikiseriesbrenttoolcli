============
Installation
============

At the command line::

    $ pip install wikiseriesbrenttoolcli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv wikiseriesbrenttoolcli
    $ pip install wikiseriesbrenttoolcli

Or, if you are using pipenv::

    $ pipenv install wikiseriesbrenttoolcli
Or, if you are using pipx::

    $ pipx install wikiseriesbrenttoolcli
